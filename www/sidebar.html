Meet Slidebox

Slidebox is a dual sidebar navigation menu template to start building your next mobile product on. It's built using CSS3, HTML, and jQuery and is compatible with PhoneGap and Cordova Build as well. You can activate the sidebar by swiping left or right too! Build your next template using this starter kit! It's easy!
